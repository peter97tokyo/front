Flexbox(유연 상자 레이아웃)은 CSS3에서 도입된 레이아웃 모듈로, 컨테이너와 아이템의 공간 배치와 정렬을 쉽게 할 수 있게 해줍니다. Flexbox는 수평 및 수직 방향으로 아이템을 배치하는 데 매우 유용합니다. 주요 속성과 함께 Flexbox의 기본 개념을 설명하고, 예제를 제공하겠습니다.

Item 3

이 예제에서는 flex-container 클래스가 Flex 컨테이너로 정의되었고, flex-item 클래스가 각 아이템을 나타냅니다. 컨테이너는 space-around를 사용하여 아이템을 주 축을 따라 균등하게 배치하고, align-items: center를 사용하여 교차 축을 따라 중앙에 정렬합니다. 아이템은 일정한 간격을 두고 배치되며, 중앙에 위치하게 됩니다.

CSS Grid는 2차원 레이아웃 시스템으로, 행과 열을 이용하여 복잡한 웹 페이지 레이아웃을 쉽게 구성할 수 있게 합니다. CSS Grid는 컨테이너와 아이템의 공간 배치를 유연하게 할 수 있는 다양한 속성을 제공합니다. 주요 속성과 함께 CSS Grid의 기본 개념을 설명하고, 예제를 제공하겠습니다.

Sidebar

이 예제에서는 grid-container 클래스가 Grid 컨테이너로 정의되었고, grid-item 클래스가 각 아이템을 나타냅니다. grid-template-columns 속성을 사용하여 3개의 열을 정의하고, grid-gap 속성을 사용하여 아이템 간의 간격을 설정합니다. 각 아이템은 grid-column 속성을 사용하여 특정 열에 배치됩니다. 예를 들어, header는 첫 번째부터 세 번째 열까지 차지하고, sidebar는 첫 번째 열에, content는 두 번째 열에, footer는 모든 열에 걸쳐 배치됩니다.
